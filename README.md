# Smart Lighting System Using LDR and PIR Sensor

This project is a smart lighting system designed in Tinkercad using Arduino Uno, an LDR, and a PIR sensor. The system automatically turns the LED ON when it is dark and motion is detected.

## Objective
To build an automatic lighting system that saves energy and works like a basic smart home light.

## Components Used
- Arduino Uno
- LDR (Photoresistor)
- PIR Sensor
- LED
- 220 ohm resistor
- 10k ohm resistor
- Jumper wires

## Working
- The LDR detects whether the environment is dark or bright.
- The PIR sensor detects motion.
- Arduino reads both sensors and controls the LED.
- LED turns ON only when it is dark and motion is detected.


## Applications
- Automatic night lights
- Smart home lighting
- Energy-saving lighting systems
- Corridor lights
- Security lighting

## Tools Used
- Tinkercad Circuits
- Arduino Uno
